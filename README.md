# Python Basics - Loops & Functions

Practice exercises covering Python loops and functions, with a few small projects for hands-on application.

## Topics Covered

**Loops**
- while, while...else
- for, for...else
- range()
- Nested loops (for dictionaries within dictionaries)
- continue, break, pass

**Functions**
- Defining functions with `def` and `return`
- Parameters & arguments
- Default parameters
- `*args` and `**kwargs`

## Mini Projects
- Ballon d'Or top 10 ranking
- Best player per club (dictionary)
- Students data (nested dictionary)
- Even numbers filter (1-100)
- Winners & losers game (user input)
- Birth years lookup using `**kwargs`

## How to Run
```bash
python PythonBasics6.py
```
Run it from a terminal, since parts of the script take user input.
